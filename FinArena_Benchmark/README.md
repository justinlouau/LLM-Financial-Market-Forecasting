# FinArena Benchmark

## Instructions

1. Download the FNSPID Dataset
2. Run the scripts in FNSPID to extract the news and price data for the desired time period
3. Create a Kaggle Notebook with the `processed_data.csv` dataset from Step 2 and ChatTS-14B model
4. Populate the Kaggle Notebook with the scripts in FinArena and run using `GPU T4 x2`
